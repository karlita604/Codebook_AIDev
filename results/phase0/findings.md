# Phase 0 — PR250 Dataset Summary

Descriptive summary of `data/04_pr250 - 04_pr250.csv`: **250 closed pull requests** authored by 5 autonomous coding agents across **128 repositories** and 61 distinct authors. Figures generated by `src/generate_phase0_figures.py`.

## Headline numbers

- Total PRs: **250**
- Unique repositories: **128** (avg 1.95 PRs/repo)
- Unique authors: **61**
- Coding agents: **5** (Devin, Copilot, OpenAI Codex, Cursor, Claude Code)
- Date range: **2024-12-24** to **2025-07-30**
- Closed without merge: **250 / 250 (100%)**
- Repos with >1 agent represented: **3 / 128**

## Submission volume over time (Figure 2)

Devin is the only agent present before May 2025. Copilot, OpenAI Codex and Cursor enter the sample from May 2025 onward; monthly volume roughly triples between April and June 2025, then eases slightly in July.

## Rejection-reason codes (Figure 3)

`manual_label` has **25 distinct numeric codes**, all populated (0 missing). The top 5 codes (Code 2, Code 4, Code 5, Code 3, Code 1) account for 68% of PRs; the remaining 20 codes are long-tail.
**Caveat:** `writing/Codebook.md` defines the qualitative rejection themes (Build Failure, CI Fail, Agent Draft, Unresolved Merge State, Incomplete Solution, Did Not Follow Instructions, Unrequested Changes, Infinite Loops, Tool Call Failures) but does not yet publish a code-number legend for all values — numeric codes are reported as-is.

## Repositories (Figure 4)

250 PRs span 128 repositories. `crewAIInc/crewAI` is the single largest source (34 PRs), followed by `airbytehq/airbyte` (13). Only 3 repositories contain PRs from more than one agent — agent assignment is almost entirely repo-exclusive in this sample.

## PR lifecycle duration (Figure 5)

Hours from `created_at` to `closed_at` are heavily right-skewed (n=250): min 0.0h, p25 0.7h, median 25.1h, p75 203.7h, p90 304.3h, max 1278.9h (mean 130.1h, pulled up by the tail).

| Agent | n | Median close time | Mean close time |
|---|---|---|---|
| Devin | 86 | 7.9d | 6.3d |
| Copilot | 80 | 22.7h | 6.7d |
| OpenAI Codex | 69 | 2.8h | 3.4d |
| Cursor | 14 | 8.9h | 2.5d |
| Claude Code | 1 | 0.7h | 0.7h |

OpenAI Codex closes fastest (median 2.8h); Devin is slowest (median 189.7h ≈ 7.9 days).

## Description length (Figure 6)

Copilot PRs carry the longest descriptions (median 320 words), OpenAI Codex the shortest (median 35 words). 5 PRs (2.0%) have an empty body.

## Data quality notes

- `merged_at` is empty for all 250 rows and `state` is "closed" for all rows — consistent with the sample being deliberately curated to closed, non-merged agent PRs for rejection-reason coding, not a random PR sample.
- `id` is unique across all rows; `(repo_id, number)` is also unique — one row per PR, no duplicates.
- `body` is empty for 5 rows (2.0%).
